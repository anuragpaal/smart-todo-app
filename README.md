# Smart Todo App 

A simple and clean Todo App built with React.

## Features
- Add tasks
- Delete tasks
- Mark as complete
- LocalStorage persistence
- Clean UI

## Tech Stack
- React
- JavaScript
- CSS
- LocalStorage

## Installation
npm install
npm run dev
